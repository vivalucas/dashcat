# 代码评审日志

## 2026-06-13：深度评审与体验优化

- **评审结论**：发现已确认问题 4 项，优化建议 2 项
- **范围**：`AppDelegate.swift`、`ClipboardManager.swift`、`ClipboardPanel.swift`、`SystemMonitor.swift`
- **已确认问题**：
  1. 剪贴板面板方向键导航完全失效，方向键触发 `tableViewSelectionDidChange` 导致意外复制并关闭面板。
  2. SQLite 数据库存储了图片的绝对路径，存在跨设备或沙盒化时的图片丢失和无法清理风险。
  3. CPU 占用率计算公式未包含 `nice` 进程，导致特定场景下统计偏低。
  4. Finder 新建文件在 AppleScript 权限被拒时缺乏引导弹窗。
- **优化建议**：
  1. 剪贴板面板底部增加 `Option+Enter` 复制纯文本的隐性高阶功能 UI 提示。
  2. 自定义保留天数时补充占位符和反馈。
- **处理决策**：全部采纳并进入实现修复阶段。

## 2026-06-08：综合项目评审

- **评审结论**：发现已确认问题 6 项，待确认风险 2 项
- **范围**：`AppDelegate.swift`、`ClipboardManager.swift`、`ClipboardPanel.swift`、README 和 `project-log`
- **已确认问题**：
  1. 防休眠 assertion 创建失败后没有回退关闭，UI 和持久化状态可能误导用户。
  2. 剪贴板历史设置为永久保留时，启动清理会跳过孤儿图片清理。
  3. 图片条目删除时先删文件再删数据库，数据库删除失败会留下失效记录。
  4. 防休眠失败回退行为与功能设计文档不一致。
  5. `05-current-status.md` 版本仍为 2.3.8，但工程配置已为 2.3.10。
  6. README 的接电 / 充电电量颜色描述仍写蓝色，与当前实现的粉色不一致。
- **待确认问题**：
  1. SQLite 单连接跨主线程和后台队列访问虽然使用 `FULLMUTEX`，但大历史 / 大图片库下仍可能造成面板卡顿。
  2. 电量 `isCharging` 缺失时用接电状态兜底，可能在部分系统状态下把“已接电”显示成“正在充电”。
- **处理决策**：已确认进入实现；体验优化包括清除历史确认选项、空状态提示、异步缩略图、Finder 文件名输入和更新检查进行中状态。

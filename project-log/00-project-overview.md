# DashCat 项目概述

## 项目名称

DashCat

## 项目背景

DashCat 是一个 macOS 菜单栏工具，目标是把几个高频但彼此独立的小工具收成一个轻量应用：

- 系统监控和猫咪动画
- 剪贴板历史管理
- 防休眠
- 外接鼠标滚轮反转
- 极简电量状态项
- Finder 当前文件夹新建文件

项目起点来自 CatMeter，后续在此基础上加入剪贴板能力，保留纯 AppKit、低依赖、低常驻开销的路线。

## 用户 / 使用场景

- 长时间使用菜单栏工具的 macOS 用户
- 需要频繁复制、回贴、筛选历史内容的人
- 想把系统监控、防休眠和电量信息收在一个图标里的人
- 需要在 Finder 中快速新建 TXT / Markdown 文件的人

## 核心功能

1. 剪贴板历史：文本和图片，搜索、Pin、过滤词、清理和图片保存开关
2. 系统监控：CPU / 内存采样，猫咪动画，紧凑数值和自定义显示
3. 防休眠：系统休眠 / 屏幕休眠 / 关闭三态切换
4. 电量：独立极简菜单栏状态项，支持接电隐藏和轻量详情菜单
5. 滚轮：仅对传统鼠标滚轮做方向反转，触控板保持系统默认
6. Finder 新建文件：从 DashCat 菜单在当前 Finder 文件夹创建 TXT / Markdown
7. 其他：开机启动、语言切换、更新检查、GitHub / 联系入口

## 核心概念

- **菜单栏猫咪**：主状态项，承载监控、防休眠和快捷入口
- **独立电量项**：可选的第二状态项，只显示极简数字
- **Clipboard Panel**：左键弹出的非激活浮窗
- **MonitorMode / DisplayMode**：监控来源和显示方式的组合
- **CaffeineMode**：防休眠状态

## 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 应用框架 | AppKit | 纯桌面应用，主窗口由菜单栏和 NSPanel 组成 |
| 语言 | Swift | 单工程、少文件、直接调用系统 API |
| 数据库 | SQLite / libsqlite3 | 剪贴板历史持久化 |
| 系统框架 | IOKit / CoreGraphics / ApplicationServices / ServiceManagement | 电量、防休眠、滚轮、开机启动 |
| 存储 | UserDefaults + Application Support | 偏好和数据库文件 |
| 第三方依赖 | 无 | 不引入 SPM 包 |

## 项目边界

- 不做独立设置窗口
- 不做 OCR、Paste Stack、模糊搜索、正则搜索
- 不做完整电池管理、健康度、历史曲线、高能耗 App 列表
- 不把 Finder 新建文件伪装成 Finder 空白处右键菜单
- 不引入第三方依赖或后台常驻服务

## 项目约束

- macOS 13 Ventura 及以上
- Apple Silicon 优先
- 本地离线运行，更新检查除外
- 低资源占用优先
- 菜单即配置入口

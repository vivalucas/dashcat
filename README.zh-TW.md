# DashCat

一個輕量的 macOS 選單列工具，把剪貼簿歷史、系統監控、防休眠和滑鼠滾輪反轉合在一隻會奔跑的小貓裡。

我的選單列上一直跑著好幾個小工具：一個看系統負載，一個管剪貼簿（Maccy），一個防休眠（Caffeine），還要額外處理外接滑鼠滾輪方向。多個圖示、多個背景處理程序，總覺得浪費。索性自己動手，從零重新設計，只保留最常用的功能：系統監控、剪貼簿管理、防休眠和滑鼠滾輪反轉。監控邏輯專為 Apple Silicon 最佳化，剪貼簿管理精簡高效，防休眠和滾輪反轉直接內建。恰到好處，不花俏。

DashCat 就是這麼來的。一隻小貓蹲在選單列，跑得快說明電腦負載高；左鍵點一下，剪貼簿歷史隨搜隨用；右鍵點一下，防休眠、滑鼠滾輪方向、監控模式、語言切換一目了然。一個圖示，多個常用工具的活全幹了。零依賴，極低資源佔用，資料全部存在本地。

## 功能

- **剪貼簿管理**
  - 左鍵點擊貓咪圖示，彈出剪貼簿面板
  - 搜尋框即時過濾
  - 單擊項目即複製，`Option + Enter` 複製純文字
  - 右鍵項目可釘選固定到頂部
  - 支援文字和圖片（圖片儲存可開關，JPEG 壓縮）
  - 歷史保留時間可自訂：7 / 14 / 30 / 90 天、永久、或 1-365 天自訂
  - 資料全部本地儲存，完全離線，不收集任何資訊

- **系統監控**
  - 預設使用緊湊數值：選單列以雙行 C/M 百分比顯示 CPU + 記憶體，盡量節省空間
  - 需要時可切換為自訂顯示：監控來源（綜合、CPU、記憶體）+ 顯示方式（動畫、動畫 + 數值）
  - 貓咪動畫速度即時反映系統壓力——跑得越快，負載越高
  - 綜合模式下自動取 CPU 和記憶體中壓力更高者驅動動畫

- **極簡電量顯示**
  - 可選的獨立選單列電量指示器，和貓咪圖示分開顯示
  - 不顯示百分號，只用極窄數字加淺藍色電量底紋，適合瀏海螢幕和擁擠選單列
  - 使用電池時，20% 及以下顯示橘色數字/描邊，10% 及以下顯示紅色數字/描邊
  - 可設定接上電源時自動隱藏，不占選單列空間
  - 使用系統電源資訊低頻更新，無動畫，保持輕量

- **防止休眠**
  - 預設色：正常，允許系統休眠
  - 藍色：阻止系統閒置休眠（螢幕仍可關閉）
  - 橘色：阻止螢幕關閉
  - 右鍵選單直接選擇，貓咪顏色即時變化

- **其他特性**
  - 11 種語言介面：中文、English、日本語、한국어、Deutsch、Français、Español、Português、Italiano、繁體中文、Русский
  - 可反轉外接滑鼠滾輪方向，觸控板繼續保持 macOS 自然捲動
  - 可在 Finder 中新建 TXT 或 Markdown 檔案，建立前會顯示目標路徑，也可臨時改選檔案夾
  - 支援登入時啟動
  - 節能設計：動畫上限 12fps，採樣間隔 5 秒，系統休眠時自動暫停
  - 零外部依賴，純 AppKit + Swift

## 系統要求

- macOS 13（Ventura）或更高版本
- Apple Silicon Mac（M 系列晶片）

## 安裝

**方式一：DMG 安裝包**

1. 進入本倉庫的 [Releases](../../releases) 頁面，下載最新的 `DashCat-<版本號>.dmg`
2. 打開 DMG，將 DashCat 拖入「應用程式」檔案夾
3. 首次啟動時，macOS 可能提示「應用程式已損毀」或「無法驗證開發者」——這是 Gatekeeper 對未經 Apple 開發者憑證簽名應用的正常攔截，應用程式本身完好。在終端機執行以下命令移除隔離標記：
   ```bash
   xattr -cr /Applications/DashCat.app
   ```
   之後雙擊即可正常啟動；或右鍵點擊 → 打開 → 彈窗中再點「打開」。

**方式二：自行編譯（無需繞過簽名）**

1. 複製本倉庫
2. 用 Xcode 打開 `DashCat.xcodeproj`
3. 在 **Signing & Capabilities** 中選擇你自己的開發者帳號
4. `⌘R` 執行，Xcode 會自動簽名

## 使用

- **左鍵點擊**貓咪圖示：打開剪貼簿歷史面板
  - 搜尋框輸入關鍵字即時過濾
  - 單擊項目複製到剪貼簿
  - `Option + Enter` 複製純文字（去除格式）
  - 右鍵項目可釘選固定/取消固定
- **右鍵點擊**貓咪圖示：打開設定選單
  - 在 Monitor 中切換緊湊數值或自訂動畫顯示
  - 在剪貼簿設定中管理圖片儲存、保留天數和清除歷史
  - 開關極簡電量顯示、設定接上電源時隱藏
  - 在 Finder 中新建檔案、反轉滑鼠滾輪、切換語言、設定登入時啟動

## 常見問題

**剪貼簿歷史儲存在哪裡？**

`~/Library/Application Support/DashCat/` 目錄下。`clipboard.db` 存文字記錄，`Images/` 檔案夾存圖片。清除歷史會同時清理這兩個位置。

**圖片儲存會佔多少空間？**

圖片以 JPEG 壓縮儲存，單張約幾百 KB。預設關閉圖片儲存，需要時在右鍵選單開啟。設有 500MB 總儲存上限，超出後會自動刪除最舊的未固定圖片。

**貓咪圖示顏色代表什麼？**

預設色 → 正常休眠狀態。**藍色** → 正在阻止系統休眠。**橘色** → 正在阻止螢幕關閉。右鍵選單可直接切換。

**為什麼反轉滑鼠滾輪需要輔助使用權限？**

DashCat 需要在系統事件流中辨識滑鼠滾輪事件並反轉方向，因此 macOS 會要求授予輔助使用權限。未授權時不會影響剪貼簿、系統監控或防休眠功能；右鍵選單會顯示提示和前往授權入口。

**為什麼新建 Finder 檔案需要允許控制 Finder？**

DashCat 只在你點擊「在 Finder 中新建檔案」時讀取 Finder 目前視窗的路徑。macOS 可能會跳出自動化權限請求，用來允許 DashCat 取得這個路徑；平時不會常駐監聽 Finder。這個入口位於 DashCat 選單，不會注入 Finder 空白處右鍵選單。

**支援 Intel Mac 嗎？**

不支援。僅提供 arm64 組建，專為 Apple Silicon 設計。

**和 Maccy / CopyClip / Amphetamine 有什麼區別？**

DashCat 將剪貼簿管理（Maccy）、系統監控、防休眠（Amphetamine / Caffeine）三者的功能最佳化合一，一個圖示、一個處理程序、零依賴。純 AppKit 實作，記憶體佔用極低。

**首次啟動時為什麼提示「應用程式已損毀」或「無法驗證開發者」？**

預建置版本未經 Apple 付費開發者憑證簽名，Gatekeeper 會顯示此提示，應用程式本身完好無損。在終端機執行 `xattr -cr /Applications/DashCat.app` 可移除隔離標記，之後即可正常啟動。如果不想執行此命令，推薦自行編譯並用個人帳號簽名。

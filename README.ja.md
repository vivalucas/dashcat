# DashCat

[中文](README.md) | [English](README.en.md) | [한국어](README.ko.md) | [Deutsch](README.de.md) | [Français](README.fr.md) | [Русский](README.ru.md)

---

macOS のメニューバーで、システムモニター、クリップボード管理、スリープ防止の 3 つのツールをずっと使っていました。3 つのアイコン、3 つのバックグラウンドプロセス——なんだか無駄に感じていました。

1 つにまとめられないか？

そうして DashCat が生まれました。メニューバーに座る 1 匹の猫——走る速度が速いほどシステム負荷が高いことを示し、左クリックでクリップボード履歴を検索でき、右クリックでスリープ防止やモニターモード、言語切替を一覧できます。1 つのアイコンで 3 つのツールの仕事をこなします。依存関係ゼロ、極低リソース、データはすべてローカルに保存。

---

## 機能

- **クリップボード管理**
  - 猫アイコンを左クリックでクリップボードパネルを表示
  - 検索ボックスでリアルタイムフィルタリング
  - クリックでコピー、`Option + Enter` でプレーンテキストとしてコピー
  - よく使う項目をピン留めして上部に固定
  - テキストと画像に対応（画像保存はトグル可能、JPEG 圧縮）
  - 保持期間をカスタマイズ：7 / 14 / 30 / 90 日、無期限、または任意の日数
  - すべてのデータはローカル保存——完全オフライン、データ収集なし

- **システムモニター**
  - 3 つのモード：複合、CPU、メモリ
  - 猫のアニメーション速度がリアルタイムのシステム負荷を反映——速く走るほど負荷が高い
  - 複合モードでは CPU とメモリのうち圧力が高い方でアニメーションを駆動
  - ステータスバーにパーセンテージ表示（オプション）

- **スリープ防止**
  - デフォルト色：通常——システムはスリープ可能
  - 青色：システムのアイドルスリープを防止（ディスプレイはオフ可能）
  - オレンジ色：ディスプレイスリープも防止
  - 右クリックメニューから直接切替——猫の色がリアルタイムに変化

- **その他**
  - 7 言語対応：English、中文、日本語、한국어、Deutsch、Français、Русский
  - ログイン時起動に対応
  - 省電力設計：アニメーション上限 12fps、サンプリング間隔 5 秒、システムスリープ時に自動一時停止
  - 外部依存ゼロ——純粋な AppKit + Swift

## 動作環境

- macOS 26（Tahoe）以降
- Apple Silicon Mac（M シリーズチップ）

## インストール

**方法 1：DMG インストーラー**

1. このリポジトリの [Releases](../../releases) ページから最新の `DashCat.dmg` をダウンロード
2. DMG を開き、DashCat をアプリケーションフォルダにドラッグ
3. 初回起動時、macOS が「アプリが破損している」または「開発元を確認できない」と表示する場合があります——これは未署名アプリに対する Gatekeeper の正常な動作です。ターミナルで以下を実行して検疫フラグを削除してください：
   ```bash
   xattr -cr /Applications/DashCat.app
   ```
   その後、ダブルクリックで通常通り起動できます。または、右クリック → 開く → ダイアログで「開く」をクリック。

**方法 2：ZIP アーカイブ**

1. [Releases](../../releases) ページから最新の `DashCat.zip` をダウンロードして解凍
2. `DashCat.app` をアプリケーションフォルダに移動
3. ターミナルで以下を実行：
   ```bash
   xattr -cr /Applications/DashCat.app
   ```

**方法 3：ソースからビルド（Gatekeeper の回避不要）**

1. このリポジトリをクローン
2. Xcode で `DashCat.xcodeproj` を開く
3. **Signing & Capabilities** で自分の開発者アカウントを選択
4. `⌘R` で実行——Xcode が自動的に署名します

## 使い方

- **左クリック**猫アイコン：クリップボード履歴パネルを開く
  - 検索ボックスに入力してフィルタリング
  - 項目をクリックしてコピー
  - `Option + Enter` でプレーンテキストとしてコピー
  - よく使う項目をピン留め
- **右クリック**猫アイコン：設定メニューを開く
  - モニターモード、スリープ防止モードの切替
  - クリップボード履歴の管理（画像保存、保持日数、履歴クリア）
  - 言語切替、パーセンテージ表示、ログイン時起動の設定

## よくある質問

**クリップボードデータはどこに保存されますか？**

`~/Library/Application Support/DashCat/` — `clipboard.db` にテキストレコード、`Images/` に画像ファイル。履歴をクリアすると両方が消去されます。

**画像の保存にどれくらい容量を使いますか？**

画像は JPEG 圧縮で保存され、1 枚あたり数百 KB です。画像保存はデフォルトでオフです。有効にした場合、合計 500 MB の上限があり、上限を超えると最も古い画像が自動削除されます。

**猫のアイコンの色は何を意味しますか？**

デフォルト → 通常のスリープ動作。**青色** → システムスリープを防止中。**オレンジ色** → ディスプレイスリープも防止中。右クリックメニューから切替できます。

**Intel Mac に対応していますか？**

いいえ。arm64 ビルドのみで、Apple Silicon 専用です。

**Maccy / CopyClip / Amphetamine とどう違いますか？**

DashCat はクリップボード管理（Maccy のような）、システムモニター、スリープ防止（Amphetamine / Caffeine のような）を 1 つの軽量メニューバーアプリに統合しています——1 つのアイコン、1 つのプロセス、依存関係ゼロ。メモリフットプリントを最小にするため純粋な AppKit で実装されています。

**初回起動時に「アプリが破損している」と表示されるのはなぜですか？**

プリビルド版は Apple Developer 証明書で署名されていないため、Gatekeeper がこのメッセージを表示します——アプリ自体は問題ありません。ターミナルで `xattr -cr /Applications/DashCat.app` を実行して検疫フラグを削除し、通常通り起動してください。この手順を避けたい場合は、ソースからビルドして自分のアカウントで署名してください。

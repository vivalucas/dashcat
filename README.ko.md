# DashCat

클립보드 기록, 시스템 모니터링, 절전 방지, 마우스 휠 반전을 달리는 고양이 하나로 합친 가벼운 macOS 메뉴 막대 앱입니다.

macOS 메뉴 막대에서 시스템 모니터, 클립보드 관리(Maccy), 절전 방지(Caffeine), 외장 마우스 휠 방향 조정 도구를 사용하고 있었습니다. 여러 개의 아이콘, 여러 개의 백그라운드 프로세스——아깝게 느껴졌습니다. 그래서 처음부터 다시 만들었고, 꼭 필요한 기능만 남겼습니다: 시스템 모니터링, 클립보드 관리, 절전 방지, 마우스 휠 반전. 모니터는 Apple Silicon에 최적화하고, 클립보드 관리는 간결하게 효율적으로, 절전 방지와 휠 반전은 기본 내장. 불필요한 기능은 없습니다.

그래서 DashCat이 탄생했습니다. 메뉴 막대에 앉아 있는 고양이——빨리 달릴수록 시스템 부하가 높다는 뜻이고, 왼쪽 클릭으로 클립보드 기록을 검색하고, 오른쪽 클릭으로 절전 방지, 마우스 휠 방향, 모니터 모드, 언어 전환을 한눈에 확인할 수 있습니다. 하나의 아이콘이 여러 일상 도구의 역할을 수행합니다. 종속성 제로, 극저 리소스, 모든 데이터는 로컬에 저장.

## 기능

- **클립보드 관리**
  - 고양이 아이콘을 왼쪽 클릭하면 클립보드 패널 열기
  - 검색창에서 실시간 필터링
  - 클릭으로 복사, `Option + Enter`로 일반 텍스트로 복사
  - 항목을 오른쪽 클릭해 상단에 고정(Pin)
  - 텍스트 및 이미지 지원 (이미지 저장 토글 가능, JPEG 압축)
  - 보관 기간 커스터마이징: 7 / 14 / 30 / 90일, 영구, 또는 1-365일 사용자 지정
  - 모든 데이터 로컬 저장——완전 오프라인, 데이터 수집 없음

- **시스템 모니터**
  - 세 가지 모니터 모드: 복합, CPU, 메모리
  - 고양이 애니메이션 속도가 실시간 시스템 부하를 반영——빨리 달릴수록 부하가 높음
  - 복합 모드에서는 CPU와 메모리 중 압력이 높은 쪽으로 애니메이션 구동
  - 두 수치 표시는 C/M 백분율을 좁은 두 줄로 표시
  - 표시 방식: 백분율 및 애니메이션, 애니메이션만, 백분율만, 두 수치

- **절전 방지**
  - 기본 색상: 정상——시스템 절전 허용
  - 파란색: 시스템 유휴 절전 방지 (디스플레이는 여전히 꺼질 수 있음)
  - 주황색: 디스플레이 절전 방지
  - 오른쪽 클릭 메뉴에서 직접 전환——고양이 색상이 실시간으로 변화

- **기타**
  - 11개 언어 지원: English, 中文, 日本語, 한국어, Deutsch, Français, Español, Português, Italiano, 繁體中文, Русский
  - 외장 마우스 휠 방향 반전, 트랙패드는 macOS 자연 스크롤 유지
  - 현재 Finder 폴더에 TXT 또는 Markdown 파일 생성, 생성 전에 대상 경로를 표시하고 다른 폴더도 선택 가능
  - 로그인 시 실행 지원
  - 절전 설계: 애니메이션 상한 12fps, 샘플링 간격 5초, 시스템 절전 시 자동 일시 정지
  - 외부 종속성 제로——순수 AppKit + Swift

## 시스템 요구 사항

- macOS 13(Ventura) 이상
- Apple Silicon Mac(M 시리즈 칩)

## 설치

**방법 1: DMG 설치 프로그램**

1. 이 저장소의 [Releases](../../releases) 페이지에서 최신 `DashCat-<version>.dmg`를 다운로드
2. DMG를 열고 DashCat을 응용 프로그램 폴더로 드래그
3. 처음 실행할 때 macOS가 "앱이 손상되었습니다" 또는 "개발자를 확인할 수 없습니다"라고 표시할 수 있습니다——이는 서명되지 않은 앱에 대한 Gatekeeper의 정상적인 동작입니다. 터미널에서 다음 명령을 실행하여 검역 플래그를 제거하세요:
   ```bash
   xattr -cr /Applications/DashCat.app
   ```
   그 후 더블클릭으로 정상적으로 실행할 수 있습니다. 또는 우클릭 → 열기 → 대화상자에서 열기 클릭.

**방법 2: 소스에서 빌드 (Gatekeeper 우회 불필요)**

1. 이 저장소를 클론
2. Xcode에서 `DashCat.xcodeproj` 열기
3. **Signing & Capabilities**에서 자신의 개발자 계정 선택
4. `⌘R`로 실행——Xcode가 자동으로 서명합니다

## 사용 방법

- **왼쪽 클릭** 고양이 아이콘: 클립보드 기록 패널 열기
  - 검색창에 입력하여 필터링
  - 항목 클릭으로 복사
  - `Option + Enter`로 일반 텍스트로 복사
  - 항목을 오른쪽 클릭해 고정/고정 해제
- **오른쪽 클릭** 고양이 아이콘: 설정 메뉴 열기
  - 모니터 모드, 절전 방지 모드 전환
  - 클립보드 기록 관리 (이미지 저장, 보관 일수, 기록 삭제)
  - 현재 Finder 폴더에 새 파일 생성, 마우스 휠 반전, 언어 전환, 표시 방식 전환, 로그인 시 실행 설정

## 자주 묻는 질문

**클립보드 데이터는 어디에 저장되나요?**

`~/Library/Application Support/DashCat/` — 텍스트 기록은 `clipboard.db`, 이미지 파일은 `Images/` 폴더. 기록을 지우면 두 위치 모두 정리됩니다.

**이미지 저장에 얼마나 많은 공간이 필요한가요?**

이미지는 JPEG 압축으로 저장되며, 장당 수백 KB입니다. 이미지 저장은 기본적으로 꺼져 있습니다. 활성화하면 총 500MB 상한이 있으며, 한도를 초과하면 고정되지 않은 가장 오래된 이미지가 자동 삭제됩니다.

**고양이 아이콘 색상은 무슨 의미인가요?**

기본 → 정상 절전 동작. **파란색** → 시스템 절전 방지 중. **주황색** → 디스플레이 절전 방지 중. 오른쪽 클릭 메뉴에서 전환할 수 있습니다.

**마우스 휠 반전에 손쉬운 사용 권한이 필요한 이유는 무엇인가요?**

DashCat은 시스템 이벤트 흐름에서 마우스 휠 이벤트를 식별하고 방향을 뒤집어야 하므로 macOS 손쉬운 사용 권한이 필요합니다. 권한이 없어도 클립보드 기록, 시스템 모니터링, 절전 방지는 계속 작동하며, 오른쪽 클릭 메뉴에 안내와 시스템 설정 바로가기가 표시됩니다.

**Finder 파일 생성에 Finder 제어 권한이 필요한 이유는?**

DashCat은 “현재 Finder 폴더에 새 파일”을 선택했을 때만 Finder의 현재 폴더 경로를 읽습니다. macOS가 자동화 권한을 요청할 수 있지만, DashCat은 백그라운드에서 Finder를 감시하지 않습니다. 이 명령은 DashCat 메뉴에 있으며 Finder의 빈 영역 우클릭 메뉴에 추가되지 않습니다.

**Intel Mac을 지원하나요?**

아니요. arm64 빌드만 제공되며, Apple Silicon 전용입니다.

**Maccy / CopyClip / Amphetamine과 어떻게 다른가요?**

DashCat은 클립보드 관리(Maccy처럼), 시스템 모니터링, 절전 방지(Amphetamine / Caffeine처럼)를 하나의 가벼운 메뉴 막대 앱으로 통합했습니다——하나의 아이콘, 하나의 프로세스, 종속성 제로. 메모리 사용량을 최소화하기 위해 순수 AppKit으로 구현되었습니다.

**처음 실행할 때 "앱이 손상되었습니다"라고 표시되는 이유는 무엇인가요?**

사전 빌드된 버전은 Apple Developer 인증서로 서명되지 않았으므로 Gatekeeper가 이 메시지를 표시합니다——앱 자체에는 문제가 없습니다. 터미널에서 `xattr -cr /Applications/DashCat.app`을 실행하여 검역 플래그를 제거한 후 정상적으로 실행하세요. 이 단계를 피하려면 소스에서 빌드하고 자신의 계정으로 서명하세요.

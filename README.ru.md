# DashCat

У меня в строке меню macOS всегда работали три инструмента: мониторинг системы, буфер обмена и предотвращение сна. Три значка, три фоновых процесса — казалось расточительством.

Нельзя ли объединить?

Так появился DashCat. Котик в строке меню — чем быстрее бежит, тем выше нагрузка на систему; левый клик открывает историю буфера обмена с мгновенным поиском; правый клик показывает предотвращение сна, режим мониторинга и смену языка с одного взгляда. Один значок выполняет работу трёх. Нулевые зависимости, минимальное потребление ресурсов, все данные хранятся локально.

## Функции

- **Менеджер буфера обмена**
  - Левый клик по значку котика открывает панель истории буфера обмена
  - Фильтрация в реальном времени через поисковую строку
  - Клик для копирования, `Option + Enter` для копирования как чистый текст
  - Закрепление часто используемых элементов (Pin) наверху
  - Поддержка текста и изображений (сохранение изображений включается, сжатие JPEG)
  - Настраиваемый срок хранения: 7 / 14 / 30 / 90 дней, навсегда или произвольный период
  - Все данные хранятся локально — полностью офлайн, сбор данных отсутствует

- **Системный монитор**
  - Три режима: Комбинированный, CPU, Память
  - Скорость анимации котика отражает нагрузку на систему в реальном времени — чем быстрее бежит, тем выше нагрузка
  - В комбинированном режиме автоматически выбирается большее значение между CPU и памятью для управления анимацией
  - Отображение процентов в строке состояния (опционально)

- **Предотвращение сна**
  - Цвет по умолчанию: нормальный — система может переходить в спящий режим
  - Синий: предотвращение перехода системы в спящий режим (экран всё ещё может выключаться)
  - Оранжевый: предотвращение также выключения экрана
  - Переключение напрямую из меню правого клика — цвет котика меняется в реальном времени

- **Прочее**
  - 7 языков: English, 中文, 日本語, 한국어, Deutsch, Français, Русский
  - Поддержка запуска при входе в систему
  - Энергоэффективность: анимация ограничена 12 fps, интервал опроса 5 сек., автоматическая пауза при переходе системы в спящий режим
  - Нулевые внешние зависимости — чистый AppKit + Swift

## Системные требования

- macOS 26 (Tahoe) или новее
- Mac на Apple Silicon (чипы серии M)

## Установка

**Способ 1: Установщик DMG**

1. Перейдите на страницу [Releases](../../releases) этого репозитория и загрузите последний `DashCat.dmg`
2. Откройте DMG и перетащите DashCat в папку «Программы»
3. При первом запуске macOS может показать «приложение повреждено» или «не удаётся проверить разработчика» — это Gatekeeper блокирует неподписанное приложение; само приложение в порядке. Выполните следующую команду в Терминале для удаления карантина:
   ```bash
   xattr -cr /Applications/DashCat.app
   ```
   После этого дважды щёлкните для обычного запуска. Или: правый клик → Открыть → нажать «Открыть» в диалоге.

**Способ 2: ZIP-архив**

1. Загрузите последний `DashCat.zip` со страницы [Releases](../../releases) и распакуйте
2. Переместите `DashCat.app` в папку «Программы»
3. Выполните в Терминале:
   ```bash
   xattr -cr /Applications/DashCat.app
   ```

**Способ 3: Сборка из исходного кода (обход Gatekeeper не требуется)**

1. Клонируйте этот репозиторий
2. Откройте `DashCat.xcodeproj` в Xcode
3. Выберите свой аккаунт разработчика в **Signing & Capabilities**
4. Запустите через `⌘R` — Xcode автоматически подпишет приложение

## Использование

- **Левый клик** по значку котика: открыть панель истории буфера обмена
  - Введите в поисковую строку для фильтрации
  - Щёлкните по элементу для копирования
  - `Option + Enter` для копирования как чистый текст
  - Закрепляйте часто используемые элементы
- **Правый клик** по значку котика: открыть меню настроек
  - Переключение режима мониторинга и предотвращения сна
  - Управление историей буфера обмена (сохранение изображений, срок хранения, очистка истории)
  - Смена языка, отображение процентов, запуск при входе в систему

## Часто задаваемые вопросы

**Где хранятся данные буфера обмена?**

`~/Library/Application Support/DashCat/` — `clipboard.db` для текстовых записей, `Images/` для файлов изображений. Очистка истории удаляет оба расположения.

**Сколько места занимают изображения?**

Изображения хранятся в JPEG (несколько сотен КБ каждое). По умолчанию сохранение изображений отключено. При включении действует общий лимит 500 МБ — при достижении лимита самые старые изображения удаляются автоматически.

**Что означают цвета котика?**

По умолчанию → нормальный режим сна. **Синий** → предотвращение перехода системы в спящий режим. **Оранжевый** → предотвращение также выключения экрана. Переключение через меню правого клика.

**Поддерживаются ли Mac на Intel?**

Нет. Только сборка arm64, разработана для Apple Silicon.

**Чем DashCat отличается от Maccy / CopyClip / Amphetamine?**

DashCat объединяет управление буфером обмена (как Maccy), системный мониторинг и предотвращение сна (как Amphetamine / Caffeine) в одном лёгком приложении для строки меню — один значок, один процесс, нулевые зависимости. Чистый AppKit для минимального использования памяти.

**Почему macOS сообщает, что приложение «повреждено» при первом запуске?**

Предварительно собранная версия не подписана сертификатом Apple Developer, поэтому Gatekeeper показывает это сообщение — само приложение в полном порядке. Выполните `xattr -cr /Applications/DashCat.app` в Терминале для удаления карантина, затем запускайте обычным способом. Чтобы избежать этого шага, соберите из исходного кода и подпишите своим аккаунтом.

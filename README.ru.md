# DashCat

Лёгкое приложение для строки меню macOS: история буфера обмена, системный мониторинг, предотвращение сна и инверсия колеса мыши в одном бегущем котике.

У меня в строке меню macOS всегда работало несколько инструментов: мониторинг системы, буфер обмена (Maccy), предотвращение сна (Caffeine) и отдельное решение для направления колеса внешней мыши. Несколько значков, несколько фоновых процессов — казалось расточительством. Поэтому я решил всё пересоздать с нуля, оставив только самое нужное: мониторинг системы, управление буфером обмена, предотвращение сна и инверсию колеса мыши. Монитор оптимизирован для Apple Silicon, буфер обмена компактный и эффективный, предотвращение сна и инверсия колеса встроены. Ничего лишнего.

Так появился DashCat. Котик в строке меню — чем быстрее бежит, тем выше нагрузка на систему; левый клик открывает историю буфера обмена с мгновенным поиском; правый клик показывает предотвращение сна, направление колеса мыши, режим мониторинга и смену языка с одного взгляда. Один значок закрывает несколько повседневных задач. Нулевые зависимости, минимальное потребление ресурсов, все данные хранятся локально.

## Функции

- **Менеджер буфера обмена**
  - Левый клик по значку котика открывает панель истории буфера обмена
  - Фильтрация в реальном времени через поисковую строку
  - Клик для копирования, `Option + Enter` для копирования как чистый текст
  - Правый клик по элементу закрепляет его наверху
  - Поддержка текста и изображений (сохранение изображений включается, сжатие JPEG)
  - Настраиваемый срок хранения: 7 / 14 / 30 / 90 дней, навсегда или произвольное значение от 1 до 365 дней
  - Все данные хранятся локально — полностью офлайн, сбор данных отсутствует

- **Системный монитор**
  - По умолчанию включены компактные значения: CPU + память в две строки C/M, чтобы экономить место в строке меню
  - Пользовательский вид позволяет выбрать источник (комбинированный, CPU, память) и стиль (анимация, анимация + значение)
  - Скорость анимации котика отражает нагрузку на систему в реальном времени — чем быстрее бежит, тем выше нагрузка
  - В комбинированном режиме автоматически выбирается большее значение между CPU и памятью для управления анимацией

- **Компактная батарея**
  - Дополнительный независимый индикатор батареи в строке меню, отдельно от котика
  - Показывает узкое число без знака процента с едва заметной зелёной заливкой для тесной строки меню
  - Может автоматически скрываться при подключении питания, не занимая место в строке меню
  - Использует системные данные питания с низкой частотой обновления, без анимации и с минимальной нагрузкой

- **Предотвращение сна**
  - Цвет по умолчанию: нормальный — система может переходить в спящий режим
  - Синий: предотвращение перехода системы в спящий режим (экран всё ещё может выключаться)
  - Оранжевый: предотвращение сна экрана
  - Переключение напрямую из меню правого клика — цвет котика меняется в реальном времени

- **Прочее**
  - 11 языков: English, 中文, 日本語, 한국어, Deutsch, Français, Español, Português, Italiano, 繁體中文, Русский
  - Инверсия колеса внешней мыши при сохранении естественной прокрутки macOS для трекпада
  - Создание TXT- или Markdown-файла в Finder с показом целевого пути и возможностью выбрать другую папку
  - Поддержка запуска при входе в систему
  - Энергоэффективность: анимация ограничена 12 fps, интервал опроса 5 сек., автоматическая пауза при переходе системы в спящий режим
  - Нулевые внешние зависимости — чистый AppKit + Swift

## Системные требования

- macOS 13 (Ventura) или новее
- Mac на Apple Silicon (чипы серии M)

## Установка

**Способ 1: Установщик DMG**

1. Перейдите на страницу [Releases](../../releases) этого репозитория и загрузите последний `DashCat-<version>.dmg`
2. Откройте DMG и перетащите DashCat в папку «Программы»
3. При первом запуске macOS может показать «приложение повреждено» или «не удаётся проверить разработчика» — это Gatekeeper блокирует неподписанное приложение; само приложение в порядке. Выполните следующую команду в Терминале для удаления карантина:
   ```bash
   xattr -cr /Applications/DashCat.app
   ```
   После этого дважды щёлкните для обычного запуска. Или: правый клик → Открыть → нажать «Открыть» в диалоге.

**Способ 2: Сборка из исходного кода (обход Gatekeeper не требуется)**

1. Клонируйте этот репозиторий
2. Откройте `DashCat.xcodeproj` в Xcode
3. Выберите свой аккаунт разработчика в **Signing & Capabilities**
4. Запустите через `⌘R` — Xcode автоматически подпишет приложение

## Использование

- **Левый клик** по значку котика: открыть панель истории буфера обмена
  - Введите в поисковую строку для фильтрации
  - Щёлкните по элементу для копирования
  - `Option + Enter` для копирования как чистый текст
  - Правый клик по элементу закрепляет или открепляет его
- **Правый клик** по значку котика: открыть меню настроек
  - Переключение Monitor между компактными значениями и пользовательской анимацией
  - Управление изображениями, сроком хранения и очисткой в настройках буфера обмена
  - Включение компактной батареи и скрытия при подключении питания
  - Создание файла в Finder, инверсия колеса мыши, смена языка, запуск при входе в систему

## Часто задаваемые вопросы

**Где хранятся данные буфера обмена?**

`~/Library/Application Support/DashCat/` — `clipboard.db` для текстовых записей, `Images/` для файлов изображений. Очистка истории удаляет оба расположения.

**Сколько места занимают изображения?**

Изображения хранятся в JPEG (несколько сотен КБ каждое). По умолчанию сохранение изображений отключено. При включении действует общий лимит 500 МБ — при достижении лимита самые старые незакреплённые изображения удаляются автоматически.

**Что означают цвета котика?**

По умолчанию → нормальный режим сна. **Синий** → предотвращение перехода системы в спящий режим. **Оранжевый** → предотвращение сна экрана. Переключение через меню правого клика.

**Почему для инверсии колеса мыши требуется разрешение Универсального доступа?**

DashCat должен распознавать события колеса мыши в системном потоке событий и менять их направление, поэтому macOS требует разрешение Универсального доступа. Без него история буфера обмена, системный мониторинг и предотвращение сна продолжают работать; меню правого клика показывает подсказку и переход в Системные настройки.

**Почему создание файла в Finder просит разрешение управлять Finder?**

DashCat читает текущую папку Finder только когда вы выбираете «Новый файл в Finder». macOS может показать запрос разрешения Автоматизации, чтобы DashCat получил этот путь; DashCat не отслеживает Finder в фоне. Команда находится в меню DashCat и не добавляется в контекстное меню пустой области Finder.

**Поддерживаются ли Mac на Intel?**

Нет. Только сборка arm64, разработана для Apple Silicon.

**Чем DashCat отличается от Maccy / CopyClip / Amphetamine?**

DashCat объединяет управление буфером обмена (как Maccy), системный мониторинг и предотвращение сна (как Amphetamine / Caffeine) в одном лёгком приложении для строки меню — один значок, один процесс, нулевые зависимости. Чистый AppKit для минимального использования памяти.

**Почему macOS сообщает, что приложение «повреждено» при первом запуске?**

Предварительно собранная версия не подписана сертификатом Apple Developer, поэтому Gatekeeper показывает это сообщение — само приложение в полном порядке. Выполните `xattr -cr /Applications/DashCat.app` в Терминале для удаления карантина, затем запускайте обычным способом. Чтобы избежать этого шага, соберите из исходного кода и подпишите своим аккаунтом.
